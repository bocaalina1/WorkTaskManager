~RO
Proiect de gestionare a unor taskuri de catre Task Manager.

Aplicatia este formata dintr-o interfata grafica care permite urmatoarele funtionalitati:
    1. Adaugare unui angajat nou
    2. Adaugare unui task simplu sau complex
    3. Vizualizarea tuturor angajatilor si a task-urilor assignate lor 
    4. Vizualizarea unor statistici referitoare la nr de taskuri terminat sau neterminate si angajatii care au un nr de ore mai mare de 40
    5. Modificarea statusului unui task.

Fiecare task are un ID, status si o variabila boolean care spune daca este assignat sau nu. In cazul in care un task complex contine alte taskuri simple/Complexe, taskurile care intra in componenta sa vor fi trecuta ca assignat in tabelul de vizualizare pentru a nu putea fi reassignate altor angajati.


~EN
Project for task management by a Task Manager. The application consists of a graphical interface that allows the following functionalities:

    1. Adding a new employee
    2. Adding a simple or complex task
    3. Viewing all employees and the tasks assigned to them
    4. Viewing statistics regarding the number of completed or uncompleted tasks and employees who have worked more than 40 hours
    5. Modifying the status of a task

Each task has an ID, status, and a boolean variable that indicates whether it is assigned or not. If a complex task contains other simple/complex tasks, the component tasks will be marked as assigned in the visualization table so they cannot be reassigned to other employees.
